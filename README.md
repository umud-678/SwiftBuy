# SwiftBuy Prototype

Bu qovluq SwiftBuy B2B sifariş platformasının frontend prototipidir.

## Fayllar

- `index.html` — fazalara keçid səhifəsi
- `phase-1-login.html` — login ekranı (Mağaza / Distribyutor)
- `phase-2-store-dashboard.html` — mağaza dashboard
- `phase-3-product-catalog.html` — məhsul kataloqu və səbət
- `phase-4-distributor-panel.html` — distribyutor sifariş paneli

## İşlətmək

Sadəcə `index.html` faylını brauzerdə açın.

Qeyd: Bu prototip backend-sizdir və yalnız UI axınını göstərir.
